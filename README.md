# Nahual

Nahual is a **real-time hand tracking recognition system** and a initial scope ML model built it in Python. The main purpose of Nahual it's to learn and identify LSM (Lengua de Señas Mexicana). It works:

* Collecting labeled hand gestures sequeneces from video input
* Training a lightweight ML classifier with those gestures
* Running a demo that recognizes gestures as characters of LSM in real time
