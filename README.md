# Nahual

Nahual is a **real-time hand tracking recognition system** and an initial-scope ML model built in Python. The main purpose of Nahual is to learn and identify LSM (Lengua de Señas Mexicana). It works:

* Collecting labeled hand gesture sequences from video input
* Training a lightweight ML classifier with those gestures
* Running a demo that recognizes gestures as characters of LSM in real time
